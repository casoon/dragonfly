Anmerkungen zum Testen:

  1. Überprüfen Sie, ob Container Queries korrekt funktionieren
  2. Testen Sie das Theme-System und farbliche Darstellung
  3. Überprüfen Sie, ob alle Komponenten korrekt gerendert werden
  4. Testen Sie auf verschiedenen Browsern für Kompatibilität

Um mit Lightning CSS zu testen, führen Sie node lightning-css-test.js aus und prüfen Sie die verarbeiteten CSS-Dateien.
